수강희망과목 등록제도 안내

 

수강희망과목 등록제도는 과목 등록기간 동안 언제든 본인의 수강신청 가능 학점만큼 수강희망과목을 등록하고,

그 결과 수강희망과목 등록인원 ≤ 학년별 또는 전체 수강제한인원(정원) 이면 모두 수강신청 처리, 수강희망과목 등록인원이 정원을 초과

하면 학년별 또는 전체 정원의 20%만 우선순위 기반 추첨을 통해 수강신청 처리됩니다.

따라서 이 기간에 일부 교과목이 사전에 수강신청 처리되면, 본 수강신청 때 신청해야하는 교과목이 줄어들어 수강신청

부담이 경감될 수 있습니다.

 

가. 시행시기: 2월초(1학기), 8월초(2학기)

 

나. 수강희망과목 처리 방법:

• 수강희망과목 등록 기간 후 교수님이 수강인원 조정기간이 있습니다. 교수님의 조정을 반영하여 수강희망과목등록

결과가 전산 처리됩니다.

 

다. 우선순위 기반 추첨제도:

• 우선순위는 1~3순위를 순서로 설정가능 (나머지 과목은 모두 4순위로 동일)

• 수강희망과목등록 메뉴에서 희망과목 내역의 순서를 Drag&Drop하여 정렬하고 순서저장 버튼 클릭하여 설정

• 희망과목 등록인원이 수강제한인원을 초과하는 경우, 학년별/전체 정원의 20%에 대해서만 추첨으로 선발되고

나머지 정원에 대해서는 초기화되어 수강신청 기간에 선착순 신청 가능

 

라. 추첨 예시:

• 전체 정원 60명인 과목A (학년별 정원 없음)에 희망과목 등록한 학생수가 1순위 10명, 2순위 40명, 3순위 20명, 4순위 100명이면

• 전체정원의 20%만 추첨에 적용하므로 12명을 수강신청내역으로 이관가능

• 1순위 10명은 모두 선발, 2순위 40명 중 2명을 무작위 추첨하여 선발, 3, 4순위 학생은 모두 탈락

마. 수강희망과목등록 결과 확인:

수강신청 처리 여부는 수강희망과목 처리 결과가 나오는 날 수강신청 사이트(sugang.korea.ac.kr)에 접속하여

수강신청 내역조회 를 통해 확인할 수 있습니다.

(또는 KUPID포털사이트 -> (상단메뉴)수업 -> 수강신청 -> 수강신청 내역조회에서도 확인 가능)
