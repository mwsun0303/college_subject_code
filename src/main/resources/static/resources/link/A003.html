강의계획안 학생 의견 수렴 안내

 

강의계획안 관련 궁금한 내용이 있거나 추가 설명이 필요한 경우 교수님께 의견을 전달드리는 제도입니다.

 

가. 시행시기: 매학기 최초 강의 계획안 공지 후 약 일주일 간

• 세부 일정은 포털-학사일정 게시판-수강신청 일정 안내 공지문 참조

 

나. 참여방법:

• 강의계획서에 대한 학생 여러분의 의견을 [포털 로그인(http://portal.korea.ac.kr/) →

(상단메뉴)수업 → 수강안내 → 학부 전공과목/ 교양.교직과목] → 해당 과목 학수번호 클릭 →

강의계획서 조회 → 조회되는 강의계획서 하단에 의견 작성 하시면 됩니다.

 

• 강의계획서가 미입력되어 있다거나, 추가적인 설명이 필요하다고 판단될 경우 구체적인 내용을 기재해주시면

해당 의견이 교수님께 전달됩니다.

 

• 교수님께서는 여러분의 의견을 참고하여 강의계획서 내용을 변경해주실 수 있습니다.
